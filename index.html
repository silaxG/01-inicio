HTML es el lenguaje de marcado de hipertexto que se encarga de definir la estrctura de tu contenido en la pagina web
se dice que es un lenguaje de marcas debido al uso de etiquetas

Anatomia de un html

<html> es el inicio del codigo de html

<head> es el encabezado del html, es decir, lo que vemos en la parte de arriba en el navegador

<body> es el cuerpo del html, es decir, lo que vemos en la pagina

<p>hola mundo</p> es la etiqueta de parrafo, es fundamental a la hora de colocar informacion para leer

</body> es el cierre del cuerpo del html

</head> es el cierre del encabezado

</html> es el cierre del codigo, es decir, el final

¿que es CSS?

CSS es es un lenguaje informatico que nos permite crear paginas web mas atractivas y agradables a la vista mediante el uso de estilos y elementos particulares o grupos de elementos de la pagina web

se complementa con HTML siendo HTML el que se encarga del contenido de la pagina web y CSS el encargado del diseño de la pagina web

¿Que es javascript

JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web

está involucrado. Es la tercera capa del pastel de las tecnologías web estándar, dos de las cuales (HTML y CSS) hemos cubierto con mucho más detalle en otras partes

Javascript se puede usar para añadir características interactivas a tu sitio web, (por ejemplo, juegos, eventos que ocurren cuando los botones son presionados o los datos son introducidos en los formularios, efectos de estilo dinámicos, animación, y mucho más).
